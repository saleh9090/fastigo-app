# Fastigo Vision

## Project Name

Fastigo

## Domain

fastigo.app

## فكرة المشروع

Fastigo هو نظام SaaS لإدارة وتتبع الطلبات والفواتير للمحلات، بحيث يستطيع صاحب المحل تسجيل طلب أو فاتورة برقم هاتف العميل، ثم يستطيع العميل متابعة حالة الطلب من التطبيق أو الرابط.

## لوحة تحكم المسؤول الرئيسي

يوجد نظام إدارة خاص بمالك منصة Fastigo، مبني باستخدام Laravel، يمكن من خلاله إدارة جميع العملاء المشتركين في المنصة.

## العملاء

المقصود بالعملاء هنا هم المحلات أو الشركات التي تشترك في Fastigo، مثل:

* مغاسل الملابس
* الخياطين
* المطاعم والمقاهي
* الورش
* المطابع
* أي محل يحتاج إلى تتبع حالة الطلب أو الفاتورة

## بيانات المحلات

يستطيع المسؤول إضافة وتعديل بيانات كل محل، وتشمل:

* اسم المحل
* رقم السجل التجاري
* اسم الشخص المسؤول
* رقم الهاتف
* البريد الإلكتروني
* نوع النشاط
* العنوان
* نوع الباقة
* تاريخ بداية الاشتراك
* تاريخ نهاية الاشتراك
* حالة الاشتراك: نشط، منتهي، موقوف

## طبيعة النظام

النظام يعمل بنموذج SaaS، بحيث يكون لكل محل حساب خاص به، وبياناته منفصلة عن باقي المحلات.

## لوحة التحكم العامة

لوحة تحكم المسؤول لا تقتصر على إدارة المحلات فقط، بل تشمل أيضاً:

* إدارة باقات الاشتراك
* إدارة مستخدمي النظام
* إدارة محتوى الموقع الإلكتروني الظاهر للجمهور
* إدارة إعدادات التطبيق
* متابعة الإحصائيات العامة
* متابعة حالة الاشتراكات والمدفوعات

## الهدف من النسخة الأولى MVP

الهدف من النسخة الأولى هو بناء نظام بسيط وعملي يسمح بـ:

* إضافة المحلات
* إدارة اشتراكات المحلات
* تسجيل الطلبات أو الفواتير
* تحديث حالة الطلب
* تمكين العميل من متابعة حالة طلبه برقم الهاتف
